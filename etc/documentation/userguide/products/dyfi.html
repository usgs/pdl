Types

dyfi
Production version of dyfi products.

Sources

US
Vince Quitoriano, National Earthquake Information Center  < vinceq@usgs.gov>

Properties

eventsource
The event network this dyfi is associated with.
eventsourcecode
The event network code this dyfi is associated with.
depth
The depth associated with this dyfi
eventtime
The origin time associated with this dyfi
latitude
The latitude associated with this dyfi
longitude
The longitude associated with this dyfi
magnitude
The first magnitude value that accompanied this dyfi.
numResp
The number of individuals completing the dyfi web dialogue for this event, obtained from the nresponses attribute of the event_data.xml file included in the product.
maxmmi
The maximum Mercalli-scale intensity for this event, obtained from the max_intensity attribute of the event_data.xml file included in the product.

Contents

event_data.xml
Data for the event, including longitude, latitude, time, depth,etc. This file supplies additional properties for the dyfi product (as above).
*_plot_attn.jpg or *_plot_atten.jpg
Plot of intensity vs. distance from epicenter, in JPG format.
*_plot_attn.ps or *_plot_atten.ps
Plot of intensity vs. distance from epicenter, in Postscript format.
*_ciim.jpg
Map of responses by city or ZIP code, in JPG format.
*_ciim.pdf
Map of responses by city or ZIP code, in PDF format.
*_ciim.ps
Map of responses by city or ZIP code, in Postscript format.
*_ciim_imap.html
Map of responses by city or ZIP code, in HTML format.
*_plot_numresp.jpg
Plot of number or responses vs. time since event origin, in JPG format.
*_plot_numresp.ps
Plot of number or responses vs. time since event origin, in Postscript format.
cdi_zip.txt
Table of intensities aggregated by city or ZIP code, in plain text format.
cdi_zip.xml
Table of intensities aggregated by city or ZIP code, in XML format.
