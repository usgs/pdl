Types

moment-tensor
focal-mechanism with Moment Tensor Parameters. See focal-mechanism.

Properties

beachball-source
The original author of this focal mechanism (network/author code).
beachball-type
Type of moment tensor. (E.g. Centroid, Mwc, Mwb)
eventtime
Input time of event. (iso8601 UTC datetime)
latitude
Input latitude of event. (degrees)
longitude
Input longitude of event. (degrees)
depth
Input depth of event. (kilometers)
derived-magnitude
Magnitude corresponding to scalar moment.
derived-magnitude-type
Type of derived magnitude.
derived-eventtime
Derived time of event. (iso8601 UTC datetime)
derived-latitude
Derived latitude of event. (degrees)
derived-longitude
Derived longitude of event. (degrees)
derived-depth
Derived depth of event. (kilometers)
scalar-moment
Moment of earthquake (Newton meters)
nodal-plane-1-strike
Strike angle of first nodal plane (degrees)
nodal-plane-1-slip
Slip angle of first nodal plane (degrees)
nodal-plane-1-dip
Dip angle of first nodal plane (degrees)
nodal-plane-2-strike
Strike angle of second nodal plane (degrees)
nodal-plane-2-slip
Slip angle of second nodal plane (degrees)
nodal-plane-2-dip
Dip angle of second nodal plane (degrees)
t-axis-azimuth
Azimuth of T axis
t-axis-plunge
Plunge of T axis
t-axis-length
Length of T axis (Newton meters)
n-axis-azimuth
Azimuth of N axis
n-axis-plunge
Plunge of N axis
n-axis-length
Length of N axis (Newton meters)
p-axis-azimuth
Azimuth of P axis
p-axis-plunge
Plunge of P axis
p-axis-length
Length of P axis (Newton meters)
tensor-mrr
Mrr component of the moment tensor (Newton meters)
tensor-mtt
Mtt component of the moment tensor (Newton meters)
tensor-mpp
Mpp component of the moment tensor (Newton meters)
tensor-mtp
Mtp component of the moment tensor (Newton meters)
tensor-mrt
Mrt component of the moment tensor (Newton meters)
tensor-mrp
Mrp component of the moment tensor (Newton meters)
percent-double-couple
Percent double couple. (percent)
sourcetime-type
Source time function, one of "box car", "triangle", "trapezoid", "unknown".
sourcetime-duration
Source time function duration
sourcetime-risetime
Source time function rise time.
sourcetime-decaytime
Source time function decay time.
num-stations-used
Number of observations used in calculation
review-status
whether moment tensor has been reviewed. "automatic" or "manual". If this property does not appear, it implies "automatic".
quakeml-publicid
The focalMechanism element publicID attribute value, when product is generated from quakeml.
eventsource
Catalog event for product
eventsourcecode
Catalog event id for product.
original-eventsource
Original catalog for product, differs from eventsource if recontributed to another catalog.
original-eventsourcecode
Original catalog event id for product, differs from eventsourcecode if recontributed to another catalog.
