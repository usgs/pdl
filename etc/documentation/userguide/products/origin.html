Types

origin
Origin products are currently read from EQXML messages.
phase-data
Similar to origin product, but underlying xml contains phase data. This type of message is usually sent in addition to origin, because the message is significantly larger.
origin-scenario
phase-data-scenario
Scenario earthquakes.

Sources

Network Code
The contributing network.

Status

UPDATE
The product is not being deleted.
DELETE
The product is being deleted.

Properties

azimuthal-gap
Azimuthal Gap of this origin in degrees.
depth
The depth associated with this origin.
depth-type
How depth was computed.
error-ellipse-azimuth
Azimuth of error ellipse major axis in degrees.
error-ellipse-intermediate
Length of error ellipse intermediate axis in meters.
error-ellipse-major
Length of error ellipse major axis in meters.
error-ellipse-minor
Length of error ellipse minor axis in meters.
error-ellipse-plunge
Plunge of error ellipse major axis in degrees.
error-ellipse-rotation
Rotation of error ellipse major axis in degrees.
evaluation-status
quakeml evaluation status of this origin.
eventsource
The event network this origin is associated with.
eventsourcecode
The event network code this origin is associated with.
eventtime
The origin time associated with this origin.
eventtime-error
Time uncertainty in seconds.
eventtype
Type of origin. Default is "earthquake".
horizontal-error
Horizontal error associated with this location in kilometers.
latitude
The latitude associated with this origin.
latitude-error
Latitude uncertainty in degrees.
longitude
The longitude associated with this origin.
longitude-error
Longitude uncertainty in degrees.
magnitude
The preferred magnitude value that accompanied this origin.
magnitude-azimuthal-gap
Azimuthal gap of stations used to compute magnitude in degrees.
magnitude-error
Any error associated with the magnitude.
magnitude-num-stations-used
Number of stations used to compute magnitude.
magnitude-source
Source of magnitude parameters, not necessarily the same as product source.
magnitude-type
The magnitude type.
minimum-distance
Distance from location to nearest station.
num-stations-used
Number of stations used to compute this location.
num-phases-used
Number of phases used to compute this location.
origin-source
Source of origin parameters, not necessarily the same as product source.
quakeml-magnitude-publicid
The magnitude element publicID attribute value, when product is generated from quakeml.
quakeml-origin-publicid
The origin element publicID attribute value, when product is generated from quakeml.
quakeml-publicid
The event element publicID attribute value, when product is generated from quakeml.
region
A region name associated with this location.
review-status
Whether this event has been reviewed by a human.
standard-error
RMS time error. Another measurement of error associated with this location.
title
Earthquake name description, when product is generated from quakeml.
version
The version associated with this origin.
vertical-error
Vertical error associated with this location.

Contents

eqxml.xml
Original EQXML message containing origin (if originally EQXML).
quakeml.xml
Original Quakeml message containing origin (if originally Quakeml).

These properties were defined at one point, but are either not used consistently or frequently.

usage
Usage information, one of "Actual" (Default), "Scenario", "Test"
scope
Scope information, one of "Public" (Default), "Internal"
depth-error
Error associated with depth.
depth-method
Method used to compute depth.
latitude-error
Latitude error associated with this location
longitude-error
Longitude error associated with this location.
num-phases-associated
Number of phases related to this event.
num-stations-associated
Number of stations related to this event.
eventtime-error
Origin time error.
preferred-flag
Boolean eqxml preferredFlag value. "true" or "false"
location-method-class
Location method category
location-method-algorithm
Specific algorithm used for location.
location-method-model
Model used for location.
error-axis-count
Number of ErrorAxes Axis elements in message
error-axis-N-azimuth
The azimuth of the Nth error-axis.
error-axis-N-plunge
The plunge of the Nth error axis.
error-axis-N-length
The length of the Nth error axis.
error-axis-N-length-unit
The length units of the Nth error axis.
eqxml-action
Action from eqxml message, if from EQXML.
eqxml-sourcekey
Source from eqxml message, if from EQXML
magnitude-azimuthal-gap
The magnitude azimuthal gap.
has-phases
"true" if phase information is embedded in the XML. There is now a separate "phase-data" product.
