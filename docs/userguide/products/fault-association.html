Types

fault-association
This product contains data about probabilities that a fault in the SCEC Community Fault Model is associated with an event..

Properties

Not applicable

Contents

faultassociations.json
[application/json] Structured JSON string containing fault association probabilities.

JSON Format

The JSON file is an extension of GEOJSON standard and contains an array of a metadata object, a result object, and a GEOJSON feature collection. The feature collection comprises of an event feature, and of one or more fault features and their association probabilites and distance from the event.

Metadata Object

process_time
The date time the fault association probabilties were created
software_version
The version of Eq2CFM that was used to compute the probabilites
cfm_version
The version of the SCEC Community Fault Model that was used to compute the probabilites

Result Object

probability
The probability in percent the earthquake is associated with this fault
title
This could be the name of an individual fault or a group of faults

Event Feature

id
Event id - this is an id in ComCat

Event Feature properties

class
Identifies what feature is being described. In this product there are 2 classes "Event" and "Fault"
mag
The magnitude of the earthquake
title
A string describing the earthquake - usually gives some geographical reference to users.
utctime
date string of origin time in UTC
version
event version given by the network operator

Fault Feature

id
SCEC CFM fault id

Fault Feature properties

distance
distance from epicenter in kilometers
probability
The probability in percent the earthquake is associated with this fault
name
SCEC CFM 5.0 Fault name and closest segment if available.
qfaultids
an array of corresponding faults ids in the QFault database.

Contents

Input_Earthquakes_to_Eval.csv
This file contains the parametric information of the event for which this product this applies. This file is an input to calculate the association probabilities.
Input_Past_Earthquakes.csv
This file contains the parametric information of past events used to calculate the association probabilities of the event described in Input_Earthquakes_to_Eval.csv.
