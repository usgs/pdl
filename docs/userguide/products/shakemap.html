Properties

eventsource
The event network the corresponding event is associated with.
eventsourcecode
The event network code the corresponding event is associated with.
map-status
The map status obtained from the grid.xml file included in the product.
REVIEWED
RELEASED
CANCELLED
event-type
The event type obtained from the grid.xml file included in the product.
ACTUAL
TEST
SCENARIO
HEARTBEAT
process-timestamp
The time this product was last updated. Obtained from the process_timestamp attribute of the grid.xml file included in the product.
event-description
A textual description of the event, such as 13.4 mi W of Wheeler Ridge, CA. Obtained from the event_description attribute of the grid.xml file included in the product.
minimum-longitude
The minimum-longitude of the ShakeMap, obtained from the lon_min attribute of the grid.xml file included in the product.
maximum-longitude
The maximum-longitude of the ShakeMap, obtained from the lon_max attribute of the grid.xml file included in the product.
minimum-latitude
The minimum-latitude of the ShakeMap, obtained from the lat_min attribute of the grid.xml file included in the product.
maximum-latitude
The maximum-latitude of the ShakeMap, obtained from the lat_max attribute of the grid.xml file included in the product.

Contents

grid.xml
The input ShakeMap grid.xml file, from which a grid of shaking intensities are extracted.
rock_grid.xml
An included ShakeMap file from which a grid of intensity/ground motions on rock are extracted.
uncertainty.xml
An included ShakeMap file from which a grid of uncertainties are extracted.
grid.xyz.xml
An included ShakeMap file from which a grid of ground motions in ASCII text format are extracted.
shape.zip
ShakeMap general-purpose GIS shape file.
*.kml
ShakeMap general-purpose GIS shape files in Google Earth format.
hazus.zip
Data files for HAZUS loss estimation.
stationlist.txt
Lists of data about seismic stations.
stationlist.xml
Lists of data about seismic stations.
metadata.txt
FGDC-compliant metadata identifying the event, along with background information concerning shakemaps, accuracy of measures and other data.
metadata.xml
FGDC-compliant metadata identifying the event, along with background information concerning shakemaps, accuracy of measures and other data.
metadata.html
FGDC-compliant metadata identifying the event, along with background information concerning shakemaps, accuracy of measures and other data.
info.xml
ShakeMap processing parameters and map summary information.
tvguide.txt
ShakeMap information for news media.
ii_thumbnail.jpg
Thumbnail map of macroseismic intensity.
intensity.jpg
A map of macroseismic intensity.
intensity.ps.zip
A map of macroseismic intensity.
pga.jpg
Map of peak ground velocity (cm/s), in JPG format.
pgv.ps.zip
Map of peak ground velocity (cm/s), in zipped Postscript format.
psa03.jpg
Map of 0.3 sec 5% damped pseudo-spectral acceleration(%g), in JPG format.
psa03.ps.zip
Map of 0.3 sec 5% damped pseudo-spectral acceleration(%g), in zipped Postscript format.
psa10.jpg
Map of 1.0 sec 5% damped pseudo-spectral acceleration(%g), in JPG format.
psa10.ps.zip
Map of 1.0 sec 5% damped pseudo-spectral acceleration(%g), in zipped Postscript format.
psa30.jpg
Map of 3.0 sec 5% damped pseudo-spectral acceleration(%g), in JPG format.
psa30.ps.zip
Map of 3.0 sec 5% damped pseudo-spectral acceleration(%g), in zipped Postscript format.
tvmap.jpg
Large format map of macroseismic intensity suitable for television broadcast, in JPG format.
tvmap.ps.zip
Large format map of macroseismic intensity suitable for television broadcast, in zipped Postscript format.
tvmap_bare.jpg
Large format undecorated map of macroseismic intensity suitable for television broadcast, in JPG format.
tvmap_bare.ps.zip
Large format undecorated map of macroseismic intensity suitable for television broadcast, in zipped Postscript format.
urat_pga.jpg
Map of PGA uncertainty, in JPG format.
urat_pga.ps.zip
Map of PGA uncertainty, in zipped Postscript format.
